#### 3. 电视连接HDMI线后，选择对应信号源进入不了机顶盒，画面显示无信号？

**答：** 请按以下6个步骤排查：

1. **检查电视HDMI接口：** 判断电视机HDMI接口是否老化或有破损
2. **更换接口或清理：** 尝试换另一个HDMI接口，或清理现有接口后重新插入；如仍不行则为电视HDMI接口故障
3. **更换HDMI线：** 换一根HDMI线尝试连接
4. **检查机顶盒状态：** 机顶盒可能处于关闭状态，按遥控器开关键确认
5. **更换电源线：** 换一根原装电源线测试
6. **电视设置问题：** 个别电视机进入信号源后需再次设置，请搜索对应电视品牌"进入信号源后黑屏/无信号"的操作方法

#### 4. 家里有网，手机可以连，但是盒子无法连WiFi？

1. **测试手机热点：** 打开手机热点，让设备连接热点，测试是否正常联网
   - 如可连热点 → 说明WiFi被限制，尝试**重启路由器**
   - 如连不上热点 → 尝试**恢复出厂设置**；若仍无法联网，联系运营人员返厂维修

> **注意：** 中文WiFi名称可以连接，但特殊字符可能无法识别，建议使用英文名称。

> **注意：** 不同型号电视机操作步骤略有差异，可搜索对应品牌"开机默认进入信号源"的操作流程。

#### 19. 请问如何判断设备通网了？

**答：** 可通过以下**4种方式**判断：

| 方式 | 判断标准 |
|:---|:---|
| **时间核对** | 显示时间是否为当前实际时间，一致则已联网 |
| **感叹号观察** | 右上角网络图标是否有感叹号，无感叹号则已联网 |
| **APP下载测试** | 下载APP时提示"下载失败"则为未联网 |
| **全网影视测试** | 进入全网影视，正下方出现"糟糕，请求没有成功"则为未联网 |

> **建议：** 联网后，将设备直接**断电重启**，可提升连接网络时效。

#### 24. 三星电视连接后不显示信号源，需要重新插显示器跟电源线才会显示？

**答：** 请按以下顺序操作：

**操作顺序：** 先切换电视信号源 → **再插HDMI显示器线** → **最后插电源线**

**原因：** 电视有50HZ/60HZ/25HZ等脉冲信号，HDMI是自适应输出，根据电视显示频率识别。国内电视基本默认50HZ，但个别型号可能不同，需调整连接顺序。

#### 27. 连上WiFi后显示"已保存"，但是上不了网？

**答：** 请按以下步骤排查：

1. **测试手机热点：** 先尝试连接手机热点，如果热点能连上，再连WiFi
   - 如还显示"已保存"但上不了网 → 需排查路由器WiFi名称
   - 中文WiFi名称一般可以连接，但如有**特殊字符**可能无法识别，建议把WiFi名称改成英文试试

2. **重启WiFi和路由器：** 将显示"已保存"的WiFi断开连接，然后重启WiFi开关，同时重启路由器

3. **修改路由器信道：** 登入路由器后台修改信道
   - 先把WiFi改成双频，分为5G和2.4G两个WiFi
   - 同时修改WiFi信道，信道改成6
   - 保存后重启路由器，再连接2.4G WiFi
